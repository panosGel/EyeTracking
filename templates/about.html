About this tool

This is an experimental tool for analysing eye-tracking data.

It is designed for analysing Tobii data files. These should be tsv files and should include all data fields.

Filtering is done by this system. Currently the settings are fixed, but I hope to change this.

You upload one or more recordings, then use these to create one or more datasets. You can then view and compare these datasets.

This tool has been developed by Andy Brown in the Web Ergonomics Lab at the University of Manchester. The code is available on Bitbucket; it is liable to change!
